<Callout type="note">Every component is a class taking `(element, options)`. Scope all queries to `this.element`, keep `init()` idempotent, and enhance markup that already works without JS.</Callout>

## Part A — Your first components (Beginner, ~1h)

- [ ] A **Counter** component: a button + a count display; clicking increments. DOM refs scoped to `this.element`
- [ ] A **Toggle** component: a button that toggles an `aria-expanded` state and shows/hides a panel
- [ ] Register both with `register({ … })`; constructor stores `element`/`options` and calls `init()`

**Acceptance:** two instances of the same component on one page work independently; no `document.querySelector` for inner elements.

## Part B — Config & communication (Intermediate, ~1h)

- [ ] A **Tabs** (or Accordion) component using **event delegation** (one listener on the root)
- [ ] Pass configuration via **`data-nc-params-…`** (valid JSON) — e.g. an initial open index — read from `this.options`
- [ ] A **"theme switch"** that dispatches a **custom event** on `window`; one or more components listen and react

**Acceptance:** config comes from the dialog/markup via params (not hard-coded); components coordinate via a custom event, not direct references.

<Reveal title="Starter component">

```js
import { register } from '@netcentric/component-loader';

class Counter {
  constructor(element, options) {
    this.element = element;
    this.options = options;       // e.g. { step: 1 }
    this.init();
  }
  init() {
    if (this.element.dataset.ncInit) return;   // idempotent
    this.element.dataset.ncInit = 'true';
    this.out = this.element.querySelector('.counter__value');
    this.n = 0;
    this.element.querySelector('.counter__btn')
      .addEventListener('click', () => this.bump());
  }
  bump() {
    this.n += this.options.step || 1;
    this.out.textContent = this.n;
  }
}
register({ Counter });
```

```html
<div class="counter" data-nc="Counter" data-nc-params-Counter='{"step":2}'>
  <button class="counter__btn">+</button>
  <span class="counter__value">0</span>
</div>
```

</Reveal>
