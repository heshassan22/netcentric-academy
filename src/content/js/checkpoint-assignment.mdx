## The task — a Tabs widget, built functionally

You're given this markup. Make the tabs work: clicking a tab shows its panel and hides the others —
using an `init`/`initAll` pattern.

```html
<div class="tabs">
  <div class="tabs__list">
    <button class="tabs__tab" data-tab="a" aria-selected="true">First</button>
    <button class="tabs__tab" data-tab="b">Second</button>
    <button class="tabs__tab" data-tab="c">Third</button>
  </div>
  <section class="tabs__panel" data-panel="a">Panel A</section>
  <section class="tabs__panel" data-panel="b" hidden>Panel B</section>
  <section class="tabs__panel" data-panel="c" hidden>Panel C</section>
</div>
```

## Steps

- [ ] Define a **`SELECTORS`** object holding the class names (root, list, tab, panel)
- [ ] Write an **`init(root)`** function that finds the tabs + panels **within `root`** and keeps them in local variables — **closures** (lesson 03)
- [ ] Wire **one delegated `click`** listener on `.tabs__list` (lesson 05) — read the clicked tab via `event.target.closest('.tabs__tab')`
- [ ] Add a `select(name)` helper that **closes over** the panels/tabs: show the panel whose `data-panel` matches, **hide** the rest (toggle `hidden`), and move `aria-selected="true"` to the active tab
- [ ] Use **modern JS** (lesson 04): `querySelectorAll` + `forEach`, a **template literal** for the `[data-panel="…"]` selector, **optional chaining** on `dataset`
- [ ] Add **`initAll()`** = `document.querySelectorAll('.tabs').forEach(init)` and run it

## Stretch
