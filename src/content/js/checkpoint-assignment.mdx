## The task — a Tabs widget, built as a class

You're given this markup. Make the tabs work: clicking a tab shows its panel and hides the others.

```html
<div class="tabs">
  <div class="tabs__list">
    <button class="tabs__tab" data-tab="a" aria-selected="true">First</button>
    <button class="tabs__tab" data-tab="b">Second</button>
    <button class="tabs__tab" data-tab="c">Third</button>
  </div>
  <section class="tabs__panel" data-panel="a">Panel A</section>
  <section class="tabs__panel" data-panel="b" hidden>Panel B</section>
  <section class="tabs__panel" data-panel="c" hidden>Panel C</section>
</div>
```

## Steps

- [ ] Write a **`class Tabs`** whose `constructor(root)` stores the root element on `this` and finds the tabs + panels inside it (scoped to `root`, never `document`)
- [ ] Wire **one delegated `click`** listener on `.tabs__list` (lesson 04) — read the clicked tab via `event.target.closest('.tabs__tab')`
- [ ] Keep `this` pointing at the instance in the handler — use a **class-field arrow** or **`.bind(this)`** (lesson 02)
- [ ] On click: show the panel whose `data-panel` matches the tab's `data-tab`, **hide** the rest (toggle `hidden`), and move `aria-selected="true"` to the active tab
- [ ] Use **modern JS** (lesson 03): `querySelectorAll` + `forEach`/spread, a **template literal** to build the `[data-panel="…"]` selector, and **optional chaining** when reading `dataset`
- [ ] Create it: `new Tabs(document.querySelector('.tabs'))`
