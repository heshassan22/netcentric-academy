A quick one that combines what you just learned: fetch a list, transform it with array methods, and
render it. Functional, plain browser JS. Uses [JSONPlaceholder](https://jsonplaceholder.typicode.com/).

The markup:

```html
<div class="todos">
  <p class="todos__status"></p>
  <ul class="todos__list"></ul>
</div>
```

## Steps

- [ ] Show `Loading…`, then `fetch` `https://jsonplaceholder.typicode.com/todos` (check `res.ok`, catch errors) — lessons 09, 10
- [ ] The API returns 200 todos — take just the **first 10** with `.slice(0, 10)`, then **count** how many are `completed` (array methods) — lesson 06
- [ ] Render each as a `<li>` (`✓` / `○` + title) using a **fragment** — lesson 11
- [ ] Put `X of 10 done` in the status
