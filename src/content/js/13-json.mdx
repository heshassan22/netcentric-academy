JSON is **text**; your code works with **objects and arrays**. Two functions bridge the two.

## `JSON.parse` — text → value

Turn a JSON string into a real object/array you can use:

```js
const text = '{"name":"Ada","langs":["JS","Java"]}';
const obj = JSON.parse(text);
obj.name;        // "Ada"
obj.langs[0];    // "JS"
```

You'll use it for data from `localStorage`, a `data-*` attribute, or any string source. (After `fetch`,
`res.json()` already parses for you — see lesson 10.)

## `JSON.stringify` — value → text

Turn an object/array back into a string — to save it, send it in a request body, or log it:

```js
JSON.stringify(obj);            // '{"name":"Ada","langs":["JS","Java"]}'
JSON.stringify(obj, null, 2);   // pretty-printed with 2-space indent
```
