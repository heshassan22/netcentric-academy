The most useful recent additions to the language — each shown **before → after** so the win is clear.

## Optional chaining `?.`

Short-circuits to `undefined` instead of throwing when part of the chain is missing.

**Before**

```js
const city = person && person.address && person.address.city;
```

**After**

```js
const city = person?.address?.city;
onChange?.();                 // call only if it exists
```

<Callout type="when">Reading data that might be missing — API responses, optional config, optional callbacks.</Callout>

## Logical assignment `||=` `??=`

Assign only when a condition holds — great for defaults and lazy init.

**Before**

```js
if (!opts.mode) opts.mode = 'auto';
this.cache = this.cache || new Map();
```

**After**

```js
opts.mode ||= 'auto';         // set if falsy
this.cache ??= new Map();     // set only if null/undefined
```

<Callout type="when">Apply a default, or lazy-init a value the first time it's needed.</Callout>

## Dynamic import & top-level await

Load code only when it's actually needed, and `await` directly at module top level.

**Before**

```js
import Chart from './chart.js';   // always in the bundle
```

**After**

```js
const { default: Chart } = await import('./chart.js');   // loaded on demand
```

<Callout type="when">Defer a heavy or rarely-used dependency out of the initial bundle — a key performance lever.</Callout>

## Promise helpers

**Before**

```js
const [a, b] = await Promise.all([p1, p2]);   // rejects if either one fails
```

**After**

```js
await Promise.allSettled([p1, p2]);   // every result, success or failure
await Promise.any([p1, p2]);          // first success wins
```

<Callout type="when">`allSettled` when you need **every** result even if some fail; `any` for **first success wins** (e.g. fallback endpoints).</Callout>

## Arrays & strings

**Before**

```js
arr[arr.length - 1];          // last item
[...arr].sort();              // copy first — .sort() mutates the original
```

**After**

```js
arr.at(-1);                   // last item
arr.toSorted();               // sorted copy, original untouched
str.replaceAll('+', ' ');     // every occurrence, no global regex
```

<Callout type="when">`.at(-1)` for the last item; the immutable `toSorted` / `toReversed` / `with` when you must not mutate shared state.</Callout>

## ES modules

**Before (CommonJS)**

```js
const { add } = require('./math.js');
module.exports = { add };
```

**After (ES modules)**

```js
import add, { name } from './math.js';
export const name = 'Ada';              // named export
export default function add(a, b) {}     // default export
```

<Callout type="when">Named exports for a set of utilities; a default export for a module's one main thing; re-exports to build a barrel (`index.js`).</Callout>
