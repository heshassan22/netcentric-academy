Five drills, beginner → advanced — a **different component each**, all in the functional style
(lessons 03 and 05). The HTML and CSS are given; you write the JS. Each starter has the selector
object and an empty `init` — fill it in, keep state in the closure.

---

## Level 1 — Toggle

- [ ] On the button click, flip the panel's `hidden` and keep `aria-expanded` in sync

<Playground
  html={`<div class="toggle">
  <button class="toggle__btn" aria-expanded="false">Details</button>
  <p class="toggle__panel" hidden>Here are the details.</p>
</div>`}
  css={`.toggle__btn { padding: .4rem .8rem; cursor: pointer; }
.toggle__panel { margin-top: .5rem; padding: .5rem; border: 1px solid #ddd; }`}
