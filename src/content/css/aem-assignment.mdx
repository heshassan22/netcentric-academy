Apply the CSS module on a **real AEM component**. A plain `card` is ready on a branch — deploy it and
style **one SCSS file**. HTL, dialog, and a page with the card placed are done for you.

```bash
# in the `code` AEM project
git fetch && git checkout assignment/css-starter-card
mvn clean install -PautoInstallSinglePackage   # deploy to author :4502
```

Open `http://localhost:4502/content/academy/en/card-assignment.html`, then style this file:

```text
ui.frontend/src/main/frontend/academy/publish/components/card/card.clientlibs.scss
```

BEM hooks (from `card.html`): `.card` · `.card__title` · `.card__text` · `.card__cta`.
