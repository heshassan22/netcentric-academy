To test one piece of code **on its own**, you swap its helpers (other modules, the network) for
**fakes** you control. A fake lets you (a) decide what it returns and (b) check how it was called.

## A fake function — `jest.fn()`

`jest.fn()` makes an empty fake function that **remembers every call**:

```js
const onSave = jest.fn();          // a fake callback

saveForm(data, onSave);            // code under test should call it

expect(onSave).toHaveBeenCalled();              // was it called at all?
expect(onSave).toHaveBeenCalledTimes(1);        // exactly once?
expect(onSave).toHaveBeenCalledWith(data);      // with the right argument?
```

You can also tell it **what to return**:

```js
const getUser = jest.fn().mockReturnValue({ name: 'Ada' });   // returns a value
const fetchUser = jest.fn().mockResolvedValue({ name: 'Ada' }); // returns a resolved promise
```

## Faking a whole module — `jest.mock`

Replace an imported module so the real one never runs:

```js
import { sendEvent } from './analytics.js';
jest.mock('./analytics.js');     // every export becomes a jest.fn()

test('tracks the click', () => {
  handleClick();
  expect(sendEvent).toHaveBeenCalledWith('cta_click');
});
```

## Faking the network — `fetch`

Don't hit a real server in a unit test. Replace `fetch` with a fake response:

```js
beforeEach(() => {
  global.fetch = jest.fn().mockResolvedValue({
    ok: true,
    json: async () => ({ id: 1, name: 'Ada' }),
  });
});

test('loads the user', async () => {
  const user = await loadUser(1);
  expect(fetch).toHaveBeenCalledWith('/api/users/1');
  expect(user.name).toBe('Ada');
});
```

<Callout type="do">Clear your fakes between tests so call counts don't carry over — `jest.clearAllMocks()` in `afterEach`, or `clearMocks: true` in the Jest config.</Callout>

<Callout type="warn">Only fake the **edges** — the network, timers, third-party modules. Never fake the thing you're actually testing; if you do, the test proves nothing. Too many fakes and your test passes while the real app is broken.</Callout>
