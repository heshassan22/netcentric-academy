## 4 — Read the value through a Sling Model

```java
// core/.../models/notice/NoticeModel.java
package biz.netcentric.academy.core.models.notice;

@Model(adaptables = Resource.class,
       defaultInjectionStrategy = DefaultInjectionStrategy.OPTIONAL)
public class NoticeModel {

    @ValueMapValue
    @Default(values = "Heads up!")
    private String message;

    public String getMessage() {
        return message;
    }
}
```

```html
<sly data-sly-use.notice="biz.netcentric.academy.core.models.notice.NoticeModel"/>
<div class="notice">${notice.message}</div>
```

<Callout type="warn">The `package` line, the folder path, and the `data-sly-use` class must match exactly. Rebuild and deploy `core` after the change.</Callout>

## 7 — Behaviour with the component loader

```js
import { register } from '@netcentric/component-loader';

function Notice(element, options) {
  // options comes from data-nc-params-Notice (JSON)

    element.classList.toggle('notice--active');
  });
}

register({ Notice });
export default Notice;
```

```scss
  &--plain { border: 0; padding: 0; }

```

AEM adds the chosen class to the component wrapper — no dialog field, no HTL change.

## 12 — Extend a Core Component

```xml
<!-- apps/academy/components/title/.content.xml -->
<jcr:root … jcr:primaryType="cq:Component"
    sling:resourceSuperType="core/wcm/components/title/v3/title"
    componentGroup="Academy"/>
```

```java
@Model(adaptables = SlingHttpServletRequest.class,
       adapters = Title.class,
       resourceType = "academy/components/title",
       defaultInjectionStrategy = DefaultInjectionStrategy.OPTIONAL)
public class TitleImpl implements Title {

    @Self @Via(type = ResourceSuperType.class)
    private Title delegate;

    @Override
    public String getText() {
        return delegate.getText() + " — Academy";   // the one override
    }

    // forward the rest to the delegate (IDE-generated)
}
```

```java
@Test
void appendsSuffix() {
    Title core = mock(Title.class);
    when(core.getText()).thenReturn("Hello");
    // inject the mock as the delegate, then:
    assertEquals("Hello — Academy", title.getText());
}
```

<Callout type="do">Reuse the Core Component wholesale, change exactly one thing through delegation, keep HTL a thin view, and let the policy + Style System carry configuration and appearance.</Callout>
