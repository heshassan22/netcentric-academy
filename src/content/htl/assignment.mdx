## Advanced — Part D

Real data is rows of records, but **HTL has no inline map/object literal** — so a list of
records is modelled with **parallel arrays read by the loop index**, a pattern you meet when
data arrives pre-shaped from Java.

<sly data-sly-test.sessionRooms="${['A1', 'B2', 'A1']}"/>

### 19. The limit

Render the price as `42.50 EUR` from `priceCents` (= 4250). Try it — then explain *why pure HTL can't*.

<Reveal>

<Callout type="warn">**Not possible in pure HTL.** There is no division (4250 / 100) — HTL has no arithmetic at all. Formatting money belongs in a Sling Model.</Callout>

```html
<!-- model.getDisplayPrice() returns "42.50" -->
${model.displayPrice} ${currency}
```

`@ format` can format a number but cannot divide it.
